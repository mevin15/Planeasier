# Quasar App

> A Quasar project

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:8080
$ npm run dev

# build for production with minification
$ npm run build

# lint code
$ npm run lint
```
## Run project

In order to use the application, you should install the back-end repo : 
https://github.com/mevin15/Backplan
